About VLSI

VLSI (Very Large Scale Integration) is the process of integrating millions of transistors into a single chip.

input a,
input b,
output y
);

assign y = a & b;

endmodule

Interview Questions

1. What is VLSI?

2. What is CMOS?

Complementary Metal Oxide Semiconductor.

3. FPGA vs ASIC?

FPGA is programmable, ASIC is fixed.

4. What is Setup Time?

Minimum time before clock edge data must be stable.

5. What is Hold Time?

Minimum time after clock edge data must be stable.
